# PRODIGY_DS_02

# Finance Fraud and Loan Risk Analysis

# Project Overview

Financial institutions process large volumes of transactions every day, making it essential to monitor customer activity, detect potential fraud, and assess loan risks effectively.

This project analyzes banking transaction data, customer profiles, and loan records to understand financial behavior and identify potential risk indicators. The dataset was cleaned and prepared using Python, followed by interactive data analysis and dashboard development in Tableau.

The objective of this project is to demonstrate how data analytics can support fraud monitoring, customer behavior analysis, and loan risk assessment within financial systems.

# Project Workflow
## Data Collection
The project uses a multi-table banking dataset that includes:
- Customer information
- Account details
- Transaction records
- Loan data
- Branch information

These datasets were merged to create a unified dataset suitable for analysis.

# Data Cleaning and Preprocessing (Python)

Data preprocessing was performed using Pandas to ensure the dataset was accurate and consistent.

### Key preprocessing steps included:

- Handling missing values using statistical imputation
- Removing duplicate and unnecessary records
- Converting columns to appropriate data types
- Merging multiple datasets into a consolidated dataset
- Creating additional features to support analysis

These steps ensured the data was reliable for further analysis and visualization.

# Exploratory Data Analysis

#### Exploratory analysis was conducted to understand patterns in:
- Customer demographics
- Transaction behavior
- Financial activity trends
- Loan distribution and potential risk indicators

These insights helped guide the development of the Tableau dashboards.

# Tableau Dashboard
Interactive dashboards were developed in Tableau to present financial insights clearly.

### Dashboard Components
- Customer Demographics Analysis
- Customer age distribution
- Account type distribution
- Customer segmentation patterns
- Transaction Behavior Analysis
- Transaction amount distribution
- High-value transactions
- Monthly transaction trends
- Loan Risk Monitoring
- Loan distribution across customers
- Loan approval patterns
- Potential loan risk indicators
- Branch Performance Overview
- Financial activity across bank branches
- Customer distribution by branch

# Key Insights

- Customer transaction activity varies significantly across demographic groups and account types.
- High-value transactions tend to concentrate within specific customer segments.
- Loan distribution patterns reveal potential risk exposure among certain customers.
- Branch-level analysis highlights differences in financial activity and customer engagement.
- Monitoring transaction patterns helps identify unusual financial behavior that may indicate fraud risk.
